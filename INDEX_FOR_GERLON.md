# Test Account Setup - Index

This directory contains all the files needed to set up and use test accounts for development and testing.

## Files Overview

1. **seed.ts** - TypeScript seed file containing test account definitions and initialization logic
2. **TEST_ACCOUNTS.md** - Detailed documentation of available test accounts and their purposes
3. **SETUP_GUIDE.md** - Step-by-step guide for setting up test accounts in your environment
4. **TEST_CREDENTIALS_QUICK_REFERENCE.txt** - Quick reference card with test account credentials

## Quick Start

1. Read `SETUP_GUIDE.md` for environment-specific setup instructions
2. Use `TEST_CREDENTIALS_QUICK_REFERENCE.txt` for quick access to test credentials
3. Refer to `TEST_ACCOUNTS.md` for detailed information about each test account
4. Run the seed script using `seed.ts` to initialize accounts in your database

## Important Notes

- These test accounts are **for development and testing only**
- Do not use these credentials in production environments
- Keep credentials secure and never commit them to public repositories
- Rotate test credentials periodically for security
